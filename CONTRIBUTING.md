# 🤝 Contributing to VidSnapAI

Thank you for your interest in contributing to **VidSnapAI** 🚀
We welcome contributions that improve the project’s quality, performance, and user experience.

---

## 📌 How to Contribute

### 1. Fork the Repository

Click the **Fork** button and clone your copy:

```bash
git clone https://github.com/your-username/VidSnapAI-Piyu24.git
cd VidSnapAI-Piyu24
```

---

### 2. Create a Branch

Use a meaningful branch name:

```bash
git checkout -b feature/your-feature-name
```

Examples:

* `feature/add-coqui-tts`
* `fix/audio-sync-bug`
* `ui/navbar-improvement`

---

### 3. Make Changes

* Follow the existing code structure
* Keep code clean and modular
* Add comments where necessary

---

### 4. Commit Changes

Use clear commit messages:

```bash
git commit -m "Add Coqui TTS integration"
```

---

### 5. Push Changes

```bash
git push origin feature/your-feature-name
```

---

### 6. Create Pull Request (PR)

* Go to your fork on GitHub
* Click **New Pull Request**
* Describe your changes clearly

---

## 🧠 Contribution Guidelines

* Keep changes **focused and minimal**
* Maintain **code readability**
* Do not break existing functionality
* Test your changes before submitting
* Follow project structure and naming conventions

---

## 🛠️ Areas You Can Contribute

* 🎬 Reel generation improvements (transitions, effects)
* 🔊 Audio processing (TTS, music, sync)
* 🧠 AI features (auto captions, enhancements)
* 🎨 UI/UX improvements
* ⚡ Performance optimization
* 🐛 Bug fixes

---

## ⚙️ Development Setup

```bash
pip install -r requirements.txt
python main.py
python generate_process.py
```

Make sure:

* FFmpeg is installed
* API keys are configured (if needed)
